# sliding-tic-tac-toe
Sliding tic tac toe game where to user goes against a computer. Number of empty tiles before sliding and board size is entered by the user before the game starts.

How to play: Run PlayGame in src with 3 arguments in command line. first is int representation of board size (nxn). Second is int of how many blank spaces on board are left before shuffling begins. Last is int of how well the computer will play (use 3 to 5, 5 is recommended but might be a little slower).

when playing user always goes first, click to place your move then computer will go. When sliding begins click the tile you want to slide and then where you want to slide it to, only adjacent sliding works to open slots. If an invalid move is done it will not be allowed and the console will print that it is an invalid move. When somebody wins it is also displayed in the console.
